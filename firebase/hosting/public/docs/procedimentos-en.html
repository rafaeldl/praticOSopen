Procedures Overview

PraticOS allows you to create custom forms and checklists to standardize your company's processes. Each procedure can have multiple fields with different input types.

Reusable Templates

Create procedure templates that can be applied to multiple Service Orders. Edit the template once and all new instances will be updated.

Checklists

Verification items with Yes/No/N/A options

Text Fields

Open fields for descriptions and observations

Photos

Attach photos directly to each procedure item

Field Types

Procedures support different field types to meet various needs:

Verification (Check)

Options: Yes, No, or N/A (Not Applicable). Ideal for inspection checklists.

Text

Free field for descriptions, observations, or custom values.

Photo

Allows attaching one or more photos to the specific item.

Manage Templates

Templates are procedure models that can be reused across multiple SOs.

How to Create a Template

  1. Go to Settings > Procedures
  2. Click on Add Procedure
  3. Define the procedure name
  4. Add the desired fields (verification, text, photo)
  5. Save the template

Item Order

Drag items to reorder them. The order defined in the template will be maintained in applied instances.

Filling Procedures

Procedures can be filled directly in the Service Order, during work execution.

Completed Procedures

After marking a procedure as completed, only Admin, Manager, and Supervisor can reopen it for editing.

How to Fill

  1. Open the Service Order
  2. Navigate to the Procedures section
  3. Select the desired procedure
  4. Fill in each field as needed
  5. Changes are saved automatically

Photos in Procedures

Each procedure item can have photos attached, allowing you to visually document the work performed.

Photo Features

  • Camera or Gallery: Capture photos in real time or select from gallery
  • Multiple Photos: Each item can have multiple photos attached
  • Viewing: Tap the thumbnail to view full screen
  • Removal: Long press to delete a photo

Frequently Asked Questions

Can I edit an already applied template?

Yes. Edits to the template only affect new instances. Already applied procedures keep the original structure.

How to reopen a completed procedure?

Only Admin, Manager, and Supervisor can reopen. In the procedures list, swipe the item and select "Reopen".

Are the photos synchronized?

Yes. All photos are uploaded to the cloud and available on all company devices.

Can I use the same procedure in multiple SOs?

Yes. A template can be applied to any number of Service Orders. Each instance is independent.
